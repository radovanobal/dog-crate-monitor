# Project: Dog Crate Monitor
Dog Crate Monitor is a device being developed to monitor the environment inside a dog crate. My personal use case is to have a screen visible in my rearview mirror, allowing me to check the temperature in the trunk where the dog crate is located. This helps me monitor conditions in the back of the car and adjust the AC as needed.

This device is based on the [Waveshare ESP32-S3 3.97-inch e-paper display](https://www.waveshare.com/esp32-s3-epaper-3.97.htm).

# Developemnt environment

The development is done in [ESP-IDF](https://docs.espressif.com/projects/esp-idf/en/stable/esp32/get-started/index.html) v5.5.3
